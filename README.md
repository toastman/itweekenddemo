# IT Weekend Demo
Demo project for IT Weekend conference

## Project stack
 * ES6
 * Webpack
 * SASS
 * Jasmine

## For developers
To run project in dev mode with live reload
`npm run dev`

Build project:
`npm run build`

Run jasmine unit tests:
`npm test`
